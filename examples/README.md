# Code Examples

This directory contains the source code for the projects and examples mentioned in the ebook.

## Structure

- **HelloAPI**: Basic Console App and Web API examples (Chapter 1-2).
- **ProductCRUD**: MVC and EF Core examples (Chapter 3-7).
- **EShop**: The complete E-Shop project (Mega Projects).

## How to run

Navigate to each folder and run:

```powershell
dotnet run
```
